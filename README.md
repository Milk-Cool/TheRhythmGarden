# TheRhythmGarden
A rhythm game with vines!

Made for [HackClub's Midnight](https://midnight.hackclub.com). 

## Building and deploying
NodeJS is required to build the website.

```
git clone https://github.com/Milk-Cool/TheRhythmGarden.git
cd TheRhythmGarden
npm i
npm run build
npm run preview
```
This ^ will build the game, put the webpage files into `dist/` and start a local webserver for previewing the page.
